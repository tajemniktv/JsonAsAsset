# Security Policy
## Supported Versions
This project is maintained on a best-effort basis and does not currently provide security updates for specific release branches.

Please use the latest available version.

## Reporting a Vulnerability
If you believe you have found a security issue related to this plugin, please open a private security report through GitHub Security Advisories or contact me directly.

Please include:
* A description of the issue
* Steps to reproduce
* Unreal Engine version
* Plugin version / commit
* Any relevant logs or screenshots

Non-security bugs should be reported through GitHub Issues.
